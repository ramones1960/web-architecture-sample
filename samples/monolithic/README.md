# monolithic — モノリス系

機能を 1 つのデプロイ単位に集約する構成のサンプル群。
小〜中規模、プロダクト立ち上げ期、チームが小さい場合に有効。

## 想定サンプル

- `layered-architecture` — プレゼン / アプリ / ドメイン / インフラの層分割
- `modular-monolith` — モジュール境界を持つ単一デプロイ
- `clean-architecture` — 依存方向を内向きに統一（ヘキサゴナル / オニオン）
